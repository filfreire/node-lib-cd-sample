# Continuous Delivery Pipeline for a node.js package

## Requirements

Node: recommended version is v6.11.2
NPM: recommended version is 3.10.10
Test Runner: Mocha - recommended version is 3.5.0

## How to say hello

```javascript
$ node
> var hello = require('./hello.js');
undefined
> hello
[Function]
> hello()
'Hello World!'
```

## How to run the test

```javascript
npm test
```
